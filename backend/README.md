# Sistema-Delivery-Marmita

* Instalar as dependencias:

```bash
yarn
```

* Rodar o projeto

```bash
yarn dev
```

* Criar uma migration:

```bash
yarn typeorm migration:create -n CreateExample
```

* Rodar as migrations

```bash
yarn typeorm migration:run
```

* Desfazer alterações da migration

```bash
yarn typeorm migration:revert
```
